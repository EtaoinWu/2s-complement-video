# Integer Representation & Two's Complement

A visual  explanation of why computers use two's complement, using math and intuitions. Watch the video and read the lecture note **[here](http://etaoinwu.github.io/2s-complement-video/)**, or alternatively, on [YouTube](https://youtu.be/-d2rYxbD4oU).

Submission for [CSCI 698](https://bytes.usc.edu/~saty/courses/cs698/f25_T/home/index.html), taught by Dr. Saty Raghavachary.

## Repository contents

* `/slides`: The presentation deck (PowerPoint `.pptx`).
* `/note`: The formal lecture notes explaining the math behind it.
  * `main.typ`: Source code written in [Typst](https://typst.app/).
* `/script`: The narrative script used for the video production.
